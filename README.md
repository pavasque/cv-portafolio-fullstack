# CV y Portafolio – Pablo Vásquez

Este repositorio corresponde al desarrollo de un currículum vitae y portafolio personal,
implementado en HTML, CSS y Bootstrap, como parte de la evaluación del módulo Full Stack.

El proyecto se encuentra desplegado utilizando **GitHub Pages** en los siguientes enlaces:

- **Portafolio:**  
  https://pavasque.github.io/cv-portafolio-fullstack/portfolio.html
